    其他操作

  • 绑定域名、添加插件、添加404页面
  • 以上就是基本搭建步骤，按照网上的教程一步步做基本没太大问题。但是网上的东西到底不能全信，什么都得自己试过了才知道。系统环境配置基本都是傻瓜式的安装，没什么大问题，但是到了Github Page 配置的时候碰到很多问题。将HEX O部署到Github上是最关键的一步，在这一步花的时间也最多。在很多教程上，都是先安装hexo git 插件

  • 完成以上步骤其实并没有花多久时间，时间都花在了一些小问题上。比如在github上添加SSH的时候，有个pub文件，一开始不懂，百度是说要用Office的publisher打开，于是又临时下载Office2016，然后破解，最后发现publisher没用，绝望之下发现其实只要用记事本就能打开，瞬间被自己蠢哭。记事本真的是一个很强大的东西。还有配置HEXO主题也花了很多时间，并且趁机好好逛了一圈HEXO的官网，发现了很多大神。我用的主题是yelee，在HEXO官网上有，上面有一个作者，会持续更新。
  • HEXO和Github官网都是全英文，这几天看的英文真的比一个学期看的都多，学好英文对敲代码是多么重要啊！！！
  • 这次搭建博客参看的教程基本都来自于CSDN、简书、segmentfault，这三个真的是好地方，基本问题都能解决，实在不行就只能百度了。虽然过程很折腾，但只要是自己喜欢的也不会怎么在乎复杂不复杂吧。也算是完成了一个小小的目标。但是这次只是参照教程在完成的，对于github的了解对git本身的学习还不够，这仿佛打开了一扇门。这几天开始研究git相关的东西，正在看《Git权威指南》。浩瀚的程序世界，有更多需要我学习。In code sky I’m walking!
